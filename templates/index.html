Voice Agent API Documentation

v{{ version }}

POST /create_agent

Create a voice agent configuration

Parameter Type Description
apiType required string Provider type: retell or vapi
voicemailMessage string Default greeting message (optional)
voiceSpeed float Speech rate (0.5-2.0)
volume retell only float Output volume (0.5-2.0)

Examples


curl -X POST {{ base_url }}/create_agent \
-H "Content-Type: application/json" \
-d '{
    "apiType": "retell",
    "voiceSpeed": 1.2,
    "volume": 0.8,
    "voicemailMessage": "Hello, please leave a message after the beep."
}'
